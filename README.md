#Espiral de Arquimedes
A finalidade principal do código é ilustrar visualmente como uma curva matemática, definida pela fórmula polar (r=a+b*theta), pode ser traduzida em uma imagem renderizada em uma janela de aplicação. 
E mostrar que a importância desse código em computação gráfica reside na sua função como um exemplo prático de geometria programática, demonstrando como equações matemáticas simples podem ser usadas para criar formas complexas e orgânicas de maneira procedural.
Por fim o código pode ser utilizado para:
**Ilustração de Conceitos Matemáticos:** Tornar visível uma função abstrata (r= a+b*theta), mostrando a relação constante entre o ângulo (theta) e a distância do centro (r).
**Fundamento de Renderização:** Servir de base para técnicas mais avançadas usadas em softwares de modelagem 3D e animação 2D.
A importância na computação gráfica é que essa técnica permite:
**Simulação de Fenômenos Naturais:** Gerar padrões realistas, como conchas, galáxias ou caminhos de furacões.
**Design Paramétrico:** Criar texturas, efeitos visuais e elementos de interface que podem ser facilmente ajustados ao mudar constantes como 'a' ou 'b'.
**Visualização de Dados:** Organizar informações cíclicas ou progressivas (como em gráficos de séries temporais) de forma intuitiva e visualmente organizada.
